GoStint is a RESTful API front-end to container based automation jobs. Integrating with Hashicorp Vault for AppRole Authentication, Response Wrapping / Cubbyhole, Transit end-to-end encryption and Secret Injection.

Goal is to be a Highly Available and Scaleable API for automation.

At this stage this project is a proof-of-concept and under development...
